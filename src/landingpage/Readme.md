# Proyecto Intermodular — Consultora Tecnológica
Este proyecto consiste en la creación de un sitio web corporativo para una consultora tecnológica, desarrollado con HTML y CSS.

---

## 📁 Estructura del proyecto
---

## 🖥️ Tecnologías utilizadas
- HTML
- CSS
- Flexbox
- Diseño adaptable
- Tipografías responsive (rem/em)
- Formularios HTML5 con validaciones

### **1. index.html**

### **2. proyecto-reservas.html**

### **3. cronologia.html**

### **4. contacto.html**

---

## 🎨 Diseño y estilos
Todos los estilos se encuentran centralizados en `style.css`.

Incluye:
- Variables CSS (colores, tamaños)
- Tipografía principal: *Arial*
- Diseño responsive para móviles, tablet y escritorio
- Transiciones suaves en botones y enlaces
- Galería de imágenes adaptable
- Cabecera con logo escalable
